## Verificación

Para ver los cambios:
1. El servidor de desarrollo debe estar corriendo: `npm run dev`
2. Abre `http://localhost:5173` en el navegador
3. Haz un hard refresh: `Ctrl + Shift + R` (Windows) o `Cmd + Shift + R` (Mac)
4. Si no se ven, limpia la caché del navegador
